- 同步的概念
    - 同步的概念
        - 在特殊情况下，控制多线程间的相对执行顺序
    - QThread类直接支持线程间的同步
        - bool QThread::wait(unsigned long time = ULONG_MAX)

        ```c
        qDebug() << "begin";

        QThread t;

        t.start(); // 创建并后动子
        t.wait(); // 等待子擬呈执行结束

        qDebug() << "end";
        ```

# 小结
- 在默认情况下，各个线程独立存在，并行执行
- 在特殊情况下，多线程的执行在时序上存在依赖
- QThread类直接支持线程间的同步（ wait()成员函数 ）
- wait() 停止当前线程的执行，等待目标线程执行结束
