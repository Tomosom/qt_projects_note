# 模型视图中的委托

- 委托的本质
    - 为视图提供数据编辑的上下文环境
    - 产生界面元素的工厂类
    - 能够使用和设置模型中的数据

- 问题
    如何自定一个委托类？

- 自定义委托时需要重写的函数
    1. createEditor
    2. updateEditorGeometry
    3. setEditorData
    4. setModelData
    5. paint (可选）
